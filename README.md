- Hero section with CTAs
- Overview of platform capabilities
- Responsive layout (HTML + CSS)
- GitHub-ready structure

## How to Use

1. Clone the repo
2. Open `index.html` in a browser
3. Deploy via GitHub Pages or Firebase Hosting
